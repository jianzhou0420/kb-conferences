# Conferences

Per-conference paper indexes for robotics / embodied-AI venues. Each conference page provides:

- **Overview** — stats, themes, entry points
- **By field** — papers grouped by topic (manipulation, VLA, locomotion, perception, planning, …)
- **By author** — alphabetical index by lead author
- **Oral / Poster** — full track listings as compact cards
- **Highlights** — hand-picked papers worth deeper reading

For deep methodological analysis of individual papers, see the topic-specific KBs ([VLA](../kb-vla/index.md), [VLN](../kb-vln/index.md), [EQA](../kb-eqa/index.md)).
